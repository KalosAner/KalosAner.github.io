#### 代码折叠

```html
<datails>
    <summary> 显示的标题 </summary>
    <p>
        ------------------ 测试折叠 ---------------------
    </p>
    <pre><code>title,value,callBack 可以缺省 </code></pre>
</datails>
```

1. `details`：折叠语法标签
2. `summary`：折叠语法展示的摘要
3. `pre`：以原有格式显示元素内的文字是已经格式化的文本
4. `code`：指定代码范例
5. `blockcode`：表示程序的代码块

## 代码块折叠

~~~html
<details>
<summary>点击展开代码</summary>
<pre><blockcode>
#include <iostream>
int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
</blockcode></pre>
</details>
~~~

</code></pre>
</details>
